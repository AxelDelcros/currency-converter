# Currency converter

## Prerequisite

- NodeJs (with npm)

## How to launch it

```sh
npm install
npm run dev
```

## How to test it (and see the coverage)

```sh

  npm install # In case you didn't installed the project before tryning to test it
  npm run test -- --coverage
```
